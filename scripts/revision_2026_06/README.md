# Revision scripts, June 2026

This directory contains public, repository-relative copies of the scripts used for the June 2026 revision artifacts. Output defaults point to `results/revision_2026_06/` rather than the local reviewer-response workspace.

- `run_fp32_precision_control.py`: FP16/FP32 precision-control run wrapper for the Phase 3 intervention design.
- `compare_fp32_original_raw_samples.py`: read-only original-vs-FP32 raw sample comparison.
- `run_mistral_exception_repeated_split.py`: CPU repeated-split analysis for the Mistral exception case.
